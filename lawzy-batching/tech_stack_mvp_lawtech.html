Placeholder convention trong template .docx: {{company_name_vi}}

batch_fill.py — core logic from docx import Document import re, copy def fill_template(template_path, data: dict) -> bytes: doc = Document(template_path) for para in doc.paragraphs: for run in para.runs: for key, val in data.items(): run.text = run.text.replace( f"{{{{{key}}}}}", str(val) ) # xử lý table cells tương tự for table in doc.tables: for row in table.rows: for cell in row.cells: for para in cell.paragraphs: for run in para.runs: for key, val in data.items(): run.text = run.text.replace( f"{{{{{key}}}}}", str(val) ) buf = BytesIO() doc.save(buf) return buf.getvalue() PDF convert — LibreOffice headless trong Docker: subprocess.run([ "libreoffice", "--headless", "--convert-to", "pdf", "--outdir", output_dir, docx_path ])

Schema SQL RLS (phân quyền) Storage
Paste thẳng vào Supabase SQL Editor create table cases ( id uuid primary key default gen_random_uuid(), name text not null, created_by uuid references auth.users(id), created_at timestamptz default now(), status text default 'draft' ); create table general_info ( id uuid primary key default gen_random_uuid(), case_id uuid references cases(id) on delete cascade, data jsonb not null, -- lưu toàn bộ master form dưới dạng JSON updated_at timestamptz default now() ); create table templates ( id uuid primary key default gen_random_uuid(), code text unique not null, name text not null, file_path text, -- Supabase Storage path field_mappings jsonb, -- {"{{company_name_vi}}": "company.name_vi"} language text default 'vi', is_required boolean default true, is_active boolean default true ); create table documents ( id uuid primary key default gen_random_uuid(), case_id uuid references cases(id) on delete cascade, template_id uuid references templates(id), output_path text, -- Supabase Storage path status text default 'pending', generated_at timestamptz );
Row Level Security — Intern chỉ thấy hồ sơ của mình -- Bật RLS alter table cases enable row level security; -- Intern thấy hồ sơ mình tạo create policy "own cases" on cases for all using (created_by = auth.uid()); -- Leader thấy tất cả (set role trong user metadata) create policy "leader sees all" on cases for all using ( auth.jwt() ->> 'role' = 'leader' ); -- Áp tương tự cho general_info và documents -- (case_id join về cases đã được RLS lọc) Set role khi tạo user: -- Supabase Dashboard → Authentication → Users -- hoặc qua Admin API: supabase.auth.admin.updateUserById(userId, { user_metadata: { role: 'leader' } })
2 bucket trong Supabase Storage templates/ ← public read, admin write vi/mau-01.docx vi/mau-02.docx en/mau-01-en.docx ... outputs/ ← private, per-case {case_id}/mau-01_CongtyABC_2025-05-06.docx {case_id}/mau-01_CongtyABC_2025-05-06.pdf {case_id}/export.zip Download signed URL (expire 1h) để tránh expose file trực tiếp: const { data } = await supabase.storage .from('outputs') .createSignedUrl(`${caseId}/export.zip`, 3600)
